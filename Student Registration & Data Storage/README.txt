Student Registration Project

Requirements:
- XAMPP / WAMP / LAMP
- PHP + MySQL

Setup:
1. Copy project folder to htdocs (XAMPP).
2. Open phpMyAdmin.
3. Import database.sql
4. Visit:
   http://localhost/student_registration_project/index.php

Features:
- HTML5 Form Validation
- CSS3 Responsive Design
- MySQL Data Storage
- SELECT query display table
